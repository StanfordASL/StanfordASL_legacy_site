Course Project

Description

Your robot is on an autonomous rescue mission! You will be expected to (1) explore an unknown environment and locate all of the objects around the world, then (2) given a rescue request from your TAs, your robot will need to autonomously rescue the objects and deliver them safely back to the home base. See this handout for details about the deliverables, baselines, extensions and rubric.

Code (starting point)

A starting point for your autonomy stack can be found in the asl_turtlebot repo. This code is merely a starting point and is no means complete. Please read the README for brief descriptions and disclaimers about the code.

Demo

The final project demo will be held over zoom on Thursday December 15, during 3:30 - 6:30 pm. Each group will be given 12-minute slots to present and demo their project. Sign up for a timeslot here (you will need to use your Stanford login to access the sheet).
